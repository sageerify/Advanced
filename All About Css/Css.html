Css

Css stands for Cascading Style Sheets. Html is the Skeleton of website. Css is the colors and beauty. JavaScript is the engine of the website.

Css is used to style the html documents. Css Describe the html element should be displyed.

Three type of css writing

1. Inline Css

In inline style sheet we write the code in the atribute in one line.

2. internal Css

In internal style sheet we write the css code in the head tag. Style tag is used to write.

3. External Css

External style sheet is the sepeate file in which we write the css. Then we link it with html file.
